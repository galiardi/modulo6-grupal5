# Modulo 6 - Grupal 5

1.  Clonar repositorio:
```sh
git clone https://github.com/galiardi/modulo6-grupal5
```
2.  Instalar dependencias:
```sh
npm install
```
3.  Levantar servidor:
```sh
npm start
```
4. Visitar <a href="http://localhost:3000" target="_blank">localhost:3000.</a>
